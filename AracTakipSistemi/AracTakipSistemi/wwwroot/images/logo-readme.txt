TURGUT ÖZAL ÜNİVERSİTESİ LOGOSU NASIL EKLENİR?

1. Turgut Özal Üniversitesi'nin resmi logosunu bu klasöre koyun
2. Logo dosyasının adı "turgut-ozal-logo.png" olmalıdır
3. Önerilen boyut: 300x300 piksel veya daha yüksek çözünürlük
4. PNG formatında olması tercih edilir (şeffaf arka plan için)

DOSYA YOLU:
AracTakipSistemi/AracTakipSistemi/wwwroot/images/turgut-ozal-logo.png

NOT:
- Eğer logo dosyası bulunamazsa, otomatik olarak üniversite ikonu gösterilir
- Logo dosyasını ekledikten sonra sayfayı yenilemeyi unutmayın
- Telif hakkı olan logoların kullanım izni alınmalıdır

LOGO KAYNAĞI:
- Turgut Özal Üniversitesi'nin resmi web sitesinden indirilebilir
- Üniversite kurumsal kimlik rehberinden temin edilebilir
